My first Rust project!

It's a snake game running in the terminal. Move by typing one of the "wasd" keys and pressing enter. I used Robert Heaton's https://robertheaton.com/2018/12/02/programming-project-5-snake/ for inspiration and guidance.

After going through about half the Rust book and the exercises on Rustlings I wanted to try doing more than just bite-sized code snippets. Tackling this is quite hard but also rewarding! I will be refactoring this code later. It's too messy atm for my liking.
